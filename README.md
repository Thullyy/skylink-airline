# SkyLink Airline Database
- A relational database system designed to manage domestic airline operations using Oracle SQL.

## About the Project
SkyLink Airline was designed to implement a reliable relational database that streamlines domestic airline operations by efficiently managing passengers, flights, bookings, aircraft, airports, staff, and payments. The system was created to improve data integrity, reduce errors such as double bookings, and support efficient airline management.

## Features
- Entity Relationship Diagram (ERD)
- Normalized database design
- Oracle SQL implementation
- Table creation scripts
- Sample data
- SQL queries and reports

## Technologies
- Oracle Database
- Oracle SQL
- SQL Developer

## How to Run
1. Open SQL Developer.
2. Connect to an Oracle database.
3. Run the SQL scripts in the provided order.
4. Execute the sample queries.
